### Webhooks vs MCP

- Learned that webhooks and MCP solve different problems.
- Webhook = event-driven notification system.
- MCP (Model Context Protocol) = standard for AI agents to use external tools and data.
- Webhooks push data automatically when an event occurs.
- MCP enables AI to query, read, and write data on demand.
- Webhooks are simple, lightweight, and ideal for automation workflows.
- MCP is useful when building AI agents that need tool access.
