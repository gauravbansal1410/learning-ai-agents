## What I built
- Thai Slack Summarizer
  - Chat Trigger → Basic LLM Chain
  - Translates Thai text + returns English bullet points only
  - Prompt engineering: used "Return ONLY bullets" to control output format

- Memory Chat Assistant
  - Chat Trigger → AI Agent → Ollama
  - Simple Memory node added — remembers context across messages
  - System Message added via Options → Add Option

## What I learned
- Prompt engineering: how wording controls AI output format
- Basic LLM Chain has no memory — AI Agent node is needed for that
- AI Agent node is more powerful than Basic LLM Chain
- Simple Memory node = AI remembers earlier messages in a conversation
- System Message = gives the AI a personality and instructions
