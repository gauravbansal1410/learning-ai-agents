## What I built
- Dynamic text summarizer in n8n
- `v1` Manual Trigger → Edit Fields → Basic LLM Chain → Output
- `v2` Chat Trigger (captures your message as chatInput) → Basic LLM Chain → Output

## What I learned
- `{{ $json.text }}` — passing data between nodes
- `{{ $json.chatInput }}` — how chat input is captured
- Chat Trigger node — gives you a real chat interface
- Edit Fields node — useful for hardcoded test data, not needed when using chat
