## What I built
- Dynamic text summarizer in n8n

## What I learned
- `{{ $json.text }}` is how nodes pass data to each other
- Edit Fields node holds input data for the workflow
- Execute Step = test one node
- Execute Workflow = run everything together
- Hardcoded input vs dynamic input — big difference
