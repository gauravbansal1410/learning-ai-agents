Local AI + n8n setup

## Goal:
Run local n8n with local AI

## Installed
- Docker Desktop
- Ollama (used llama 3)
- n8n

## Commands

Start Docker:
open -a docker

Check Docker:
docker ps
docker info

Start n8n
docker start -ai n8n

Run n8n
http://localhost:5678/

Stop n8n
Ctrl + C

Stop Docker
pkill docker

Checking ollama
ollama run llama3
