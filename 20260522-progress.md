Local AI + n8n setup

## Goal:
Run local n8n with local AI

## Installed
- Docker Desktop
- Ollama (used Llama 3)
- n8n

## Learnings

- Execute Step != Execute Workflow
- Docker containers need persistence
- Ollama runs separately from n8n
- In n8n to configure ollama, need to configure at `http://host.docker.internal:11434`

## Commands

Start Docker:
```bash 
open -a docker
```

Check Docker:
```bash
docker ps
docker info
```

Start n8n
```bash
docker start -ai n8n
```

Run n8n
```bash
http://localhost:5678/
```

Stop n8n
```bash
Ctrl + C
```

Stop Docker
```bash
pkill docker
```

Checking ollama
```bash
ollama run llama3
```

Exit ollama
```bash
/bye
```
