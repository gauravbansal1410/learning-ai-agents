## Key Learnings
- Google OAuth rejects raw IP addresses as redirect URIs — needs a proper domain name
- DuckDNS provides free subdomains instantly, no credit card needed (`<duckdns-domain>`)
- Docker needs `rm` + `run` (not just restart) to update environment variables
- n8n Wait node pauses the whole workflow once — does NOT pace per-item calls
- n8n Batch Processing in LLM Chain node also does NOT properly delay between model calls
- **Loop Over Items** (batch size 1) + Wait node inside loop is the correct way to rate limit per-item API calls
- `Always Output Data` on HTTP Request node ensures empty responses still pass an item downstream
- `Object.keys($json).length` is the right way to detect empty items in n8n IF nodes
- Gemini free tier: gemini-flash-lite has 15 RPM, gemini-3.5-flash has only 5 RPM

## Key Learnings
- n8n Variables UI is a paid feature — Docker env vars are the free alternative
- $env expressions show "access denied" in preview but work fine at runtime
- URL fields need Expression mode (not Fixed) to evaluate $env variables
- gemini-3.5-flash free tier = 20 req/day — too low for 30 posts
- gemini-flash-lite free tier = 1500 req/day — use this instead
