## How to set up Claude Code on any new Mac from scratch

### Step 1: Install Claude Code
`curl -fsSL https://claude.ai/install.sh | sh`

Verify it worked:
`claude --version`

### Step 2: Set up dual accounts
Open your shell config file:
`nano ~/.zshrc`

Add these two lines at the bottom:
`alias claude-p='CLAUDE_CONFIG_DIR=~/.claude-personal claude'`
`alias claude-w='CLAUDE_CONFIG_DIR=~/.claude-work claude'`

Save: Ctrl+X → Y → Enter

Reload it:
source ~/.zshrc

### Step 3: Log into each account
claude-p   ← log in with personal Claude account
claude-w   ← log in with work Claude account

Verify with /status inside Claude Code

### Step 4: Confirm accounts
claude-p → should show personal email + Claude Pro
claude-w → should show work email + Claude Team

## Key concepts learned
- Terminal: text-based interface to control your Mac
- `mkdir -p` creates nested folders in one command, 
  won't error if folder exists
- `~/.zshrc` special file Mac reads every time 
  Terminal opens — used for aliases and shortcuts
- `alias` a shortcut command you define yourself
- `CLAUDE_CONFIG_DIR` tells Claude Code which account 
  config to use
- `/status` command inside Claude Code to verify which account you're on
- `MCP servers`: connections to external tools
