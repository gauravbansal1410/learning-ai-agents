## Terminal commands
- Print/display current working directory/folder `pwd`
- List files in current folder `ls`
- Move into a folder `cd foldername`
- Go up one folder `cd ..`
- Go to home folder `cd ~`
- Create a new folder `mkdir newfolder`
- Create an empty file `touch newfile.txt`
- Open current folder in finder `open .`
- clear screen `clear`

Don't run any command with `rm`, `sudo` or `-rf`. Check first.

Cancel any command `CTRL + C`
Pause any command `CTRL + Z`. To resume, type `fg`
