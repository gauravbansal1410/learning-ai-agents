## Key Learnings
- n8n needs `N8N_HOST`, `N8N_PROTOCOL`, `WEBHOOK_URL` env vars set correctly for OAuth to work
- Google OAuth rejects raw IP addresses as redirect URIs — needs a proper domain
- DuckDNS provides free subdomains instantly, no credit card needed
- Docker `rm` + `run` is needed to update env variables (can't just restart)
